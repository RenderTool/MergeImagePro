# Merge Image Pro

一个功能强大的图片合并工具，支持多张图片的网格布局、样式设置和导出。

## 功能特点

- ✅ 支持多张图片上传和管理
- ✅ 网格布局配置（行数、列数）
- ✅ 样式设置（间距、边距、边框、圆角、背景色）
- ✅ 实时预览和缩放控制
- ✅ 支持多种导出格式（JPEG、PNG、WebP）
- ✅ 响应式设计，支持手机端
- ✅ 撤销/重做操作
- ✅ 图片多选管理

## 技术栈

- Vue 3 + TypeScript
- Pinia 状态管理
- Vite 构建工具
- Canvas API 渲染


## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 生产构建

```bash
npm run build
```

## 部署

项目支持通过 GitHub Actions 自动部署到 GitHub Pages。

### 本地构建预览

```bash
npm run build
npm run preview
```

## 使用指南

1. **上传图片**：点击添加图片按钮或拖拽图片到上传区域
2. **配置布局**：设置行数、列数、间距等参数
3. **调整样式**：设置边框、圆角、背景色等样式
4. **预览效果**：实时查看合并效果
5. **导出图片**：选择格式和质量，点击导出按钮
